It does not require JS.

Buttons

By default buttons are styled. You can style a link as button with the .btn class and, optionally, .rounded class for rounded corners. Use standard html colors names like .red to set the background. Foreground and borders are adjusted automatically. You can use the .pulse class to catch attention of the users. Text color is adjusted automatically based on background color.

.btn.red .btn.navy .btn.green .btn.orange.rounded .btn.white.pulse
.btn.small .btn (normal) .btn.large.purple.rounded
.btn.small.oval .btn.oval (normal) .btn.large.oval.olive

Icons

Progress

Forms

Stupid.css does not require any special structure for forms and does not require special classes. Form elements are syled as below by default. You can use the grid elements (.quarter and .half) to align the form elements.

GRID

Stupid.css has a simple grid system with only these classes: .container, .quarter, .third, .half, .twothirds and .threequarters. You can put as many quarters, halfs, etc. into a container. They will flow accordingly. Stupid.css prevents them from squeezing below 200px, unless you use the class .compressible.
